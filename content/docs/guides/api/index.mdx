# API Overview

- **Base URL:** `https://api.example.com/v1`
- **Auth:** Bearer tokens (see **API Keys**)
- **Formats:** JSON requests/responses

## Conventions
- Pagination via `?page` and `?limit`
- Errors return `{ error: { code, message } }`
