# Prescription

This API provides methods to **create prescriptions**.

---

## Create Prescription

### Endpoint Overview

| Field | Details |
| --- | --- |
| **Endpoint** | `/api/v2/createPrescription` |
| **Method** | `POST` |
| **Content-Type** | `application/json` |
| **Request Body** | JSON object containing the fields required to create a patient prescription. |

### Example JSON Request

```json
{
    "patient_id": "66df09779557f",
    "products": [
        {
            "prod_id": 1683,
            "qty": 1,
            "sig": "Use as directed.",
            "doc_note": "Use once daily as directed.",
            "refills": 2
        }
    ],
    "ship_method": "overnight",
    "ship_to": "physician",
    "bill_to": "physician"
}
```

### Example JSON Response

```json
{
    "prescriptionID": 678500
```
