---

## Access Token

- Users must first request an access token from the API using their **username**, **password**, and **API secret**.
- The access token is **valid for 24 hours**.
- The access token must be included as a **Bearer Token** in all requests.

---

## Endpoint Overview

| Field         | Details                       |
|---------------|------------------------------|
| **Endpoint**  | `/api/v2/accessToken`         |
| **Method**    | `POST`                        |
| **Content-Type** | `application/x-www-form-urlencoded` |
| **Request Body** | The form data that includes the required fields to obtain your authentication token. |

---

## Parameters

| Key       | Type   | Description                            |
|-----------|--------|----------------------------------------|
| `username`| string | Example Username                       |
| `password`| string | Example Password                       |
| `secret`  | string | Example 32-character secret string     |

---

## Example Request (`x-www-form-urlencoded`)

| Key       | Value                                    |
|-----------|------------------------------------------|
| username  | `johndoe`                                 |
| password  | `qwerty123`                               |
| secret    | `Xh7pM2Z5Tq9Vw3Lb8jN0Yk4Rf6G1QdCs`        |
