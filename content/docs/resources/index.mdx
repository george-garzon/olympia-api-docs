You can import the official Postman collection to quickly get started testing API endpoints.

**Download:** [Olympia API Postman Collection](https://example.com/path-to-your-postman-collection.json)

To import into Postman:
1. Open Postman.
2. Click **Import** in the top-left corner.
3. Select **Link** or **File**.
4. Paste the URL above or upload the `.json` file.
5. Click **Import**.

## ðŸ”‘ Authentication Setup

Before running any request:
1. Open the **Authorization** tab in the Postman collection.
2. Set **Type** to **Bearer Token**.
3. Run the `Authentication` request to retrieve your token.
4. Copy the token and paste it in the Bearer Token field for the collection.
